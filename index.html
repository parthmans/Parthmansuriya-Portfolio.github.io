Insightful Exploration of Superstore Sales Trends

Description: In this project, I conducted an in-depth exploratory data analysis on the Superstore Sales dataset from Kaggle, aiming to uncover key sales and profit trends from 2014 to 2017. Data cleaning and preprocessing were meticulously performed using Microsoft Excel. Profit trends were analyzed across states, regions, categories, sub-categories, and timelines (years and quarters). Significant insights included identifying top-performing and least-performing products and the most profitable customers. The findings were presented through two interactive Power BI dashboards, focusing on yearly and quarterly data. This project demonstrates the effective use of SQL, Excel, and Power BI in data analysis and strategic decision-making.

Tools Used: Power BI, SQL and Microsoft Excel

Dataset: Kaggle.com

Skills Demonstrated: Data Analysis, Data Cleaning, Data Preprocessing, Data Visualization, Business Intelligence, Exploratory Data Analysis, Sales and Profit Trends, Analytical Thinking, Interactive Dashboard Creation
